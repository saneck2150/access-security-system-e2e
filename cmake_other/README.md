## Adding New Dependencies

1. Add `FetchContent_Declare` in `third_party.cmake`
2. Call `FetchContent_MakeAvailable`
3. Link with `target_link_libraries` in relevant `CMakeLists.txt`

```cmake
FetchContent_Declare(my_lib
  GIT_REPOSITORY https://github.com/example/my_lib.git
  GIT_TAG        v1.0.0
  SOURCE_DIR     "${CMAKE_SOURCE_DIR}/third_party/my_lib"
)
FetchContent_MakeAvailable(my_lib)
```

## Build Location

Dependencies are fetched to `third_party/` and built in `build/_deps/`.
