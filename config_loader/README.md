# config_loader

YAML configuration loading for the access-security system.

## Overview

Parses `access_security.yaml` and populates typed configuration structs used throughout the application.

## Configuration Sections

| Section | Struct | Purpose |
|---------|--------|---------|
| `frame_handler` | `FrameHandlerConfig` | Anti-replay, timing, binding rules |
| `storage` | `StorageConfig` | SQLite database path |
| `admin` | `AdminConfig` | HTTP API settings |
| `key_management` | `KeyManagementYaml` | Key version, master key path |

## YAML Structure

```yaml
frame_handler:
  anti_replay_enabled: true       # Enable replay attack protection
  replay_window_size: 256         # Sequence numbers to track
  max_ct_len: 4096                # Maximum ciphertext length
  max_skew_ms: 120000             # Clock drift tolerance (2 min)
  allow_previous_key_version: true
  enforce_reader_door_binding: true

key_management:
  current_key_version: 1          # Active key version
  allow_previous_key_version: true
  master_key_path: "secrets/master_key.hex"

storage:
  sqlite_path: "data/access.db"

admin:
  bind_host: "0.0.0.0"            # Network interface
  port: 8080                      # HTTP port
  admin_token: "admin"            # Bearer token (change in prod!)
  max_upload_bytes: 20000000      # ~20MB upload limit
  max_events: 1024                # Event buffer size
```

## Usage

```cpp
#include <config_loader/config.hpp>

auto cfg = config_loader::loadFromYaml("config/access_security.yaml");

// Access configuration values
std::cout << "Port: " << cfg.admin.port << "\n";
std::cout << "DB: " << cfg.storage.sqlitePath << "\n";
std::cout << "Key version: " << cfg.keyManagement.currentKeyVersion << "\n";
```

## Validation

The loader performs validation during parsing:

| Check | Error |
|-------|-------|
| `replay_window_size == 0` when `anti_replay_enabled` | `runtime_error` |
| YAML parse failure | `runtime_error` with details |
| Missing keys | Uses default values (no error) |
